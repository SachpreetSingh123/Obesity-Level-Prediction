# Obesity Level Prediction Using Machine Learning
This project predicts obesity levels using machine learning techniques based on lifestyle, eating habits, and physical condition data.

The project includes:
- Classification
- Regression
- Clustering
- Feature Engineering
- Data Visualization

#Dataset
Dataset Size:
- 2,111 records
- 17 features

Features include:
- Age
- Height
- Weight
- Physical Activity
- Water Intake
- Eating Habits
- Transportation Mode

#Algorithm Used
Classification:
- Logistic Regression
- Decision Tree
- Random Forest
- KNN
- SVM

Regression:
- Linear Regression
- Decision Tree Regressor
- Random Forest Regressor
- KNN Regressor
- SVR

Clustering:
- K-Means
- Hierarchical Clustering
- DBSCAN

#Results
Best Classification Model:
Random Forest Classifier

Accuracy: 94.8%

Key Findings:
- Weight was the most important feature
- Physical activity strongly influenced obesity levels
- Ensemble methods outperformed individual models
